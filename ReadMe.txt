# Dashboard Interactivo: Crecimiento del PIB en América

## Descripción

Este proyecto consiste en el desarrollo de un dashboard interactivo en Streamlit para visualizar y analizar el crecimiento anual del Producto Interno Bruto (PIB) de los países del continente americano.

La aplicación permite seleccionar un país desde un panel lateral, visualizar su evolución mediante un gráfico de líneas, filtrar la información por un rango de años y consultar indicadores resumen, como el crecimiento promedio del país seleccionado, el promedio global y la diferencia entre ambos.

El objetivo del proyecto es facilitar el análisis exploratorio de los datos mediante una interfaz intuitiva y dinámica.

## Tecnologías utilizadas

* Python
* Streamlit
* Pandas
* NumPy
* Matplotlib

## Funcionalidades

* Selección de países de América.
* Visualización del crecimiento anual del PIB mediante un gráfico de líneas.
* Filtro por rango de años.
* Indicadores del crecimiento promedio del país seleccionado.
* Comparación con el crecimiento promedio global.
